# 业绩统计摘要仪表盘 Tab 设计

日期：2026-06-17

## 1. 背景

当前业绩统计看板已经形成“主维度汇总列表 + 抽屉多维拆解”的报表结构。用户可以按某个主维度查看列表，也可以进入单行查看拆解和明细。

缺口在于：用户无法直接看到当前筛选结果下的整体总计。例如列表里展示了多个咨询师、部门、渠道或日期时，用户还需要一个总览入口来查看全部数据的关键指标合计。

这个需求不只是给现有列表增加一行总计。摘要统计更接近经营总览，后续还可能承载排名图、柱状图、趋势图等图表。因此本次设计把页面从单一报表升级为“仪表盘 / 报表”两个内容模式。

## 2. 目标

- 新增一个内容 Tab 区域，包含“仪表盘”和“报表”。
- “仪表盘”展示当前筛选结果下的 10 个总览指标。
- “报表”保留当前主维度汇总列表、业绩拆解和业绩明细能力。
- 顶部筛选项同时影响“仪表盘”和“报表”。
- 将“主维度”从全局筛选区移到“报表”Tab 内，使它表达为报表分组方式，而不是全局数据筛选条件。

## 3. 不做范围

- 不新增排名图、柱状图、趋势图等具体图表。
- 不做指标自定义配置。
- 不做看板保存、权限体系或真实接口接入。
- 不改变现有指标口径。
- 不改变现有抽屉拆解规则。

## 4. 页面结构

页面调整为两层：

1. 顶部全局筛选区
2. 下方内容 Tab 区

### 4.1 全局筛选区

全局筛选区只保留会限定数据范围的条件，例如：

- 统计时间
- 客户统计范围
- 部门 / 咨询师
- 渠道分类 / 渠道
- 项目分类 / 项目
- 城市 / 机构

这些筛选项决定“进入统计的数据是哪一批”。筛选结果同时作用于“仪表盘”和“报表”。

“主维度”不再放在全局筛选区。主维度不会改变统计数据范围，只决定报表如何分组展示。

### 4.2 内容 Tab

新增两个 Tab：

| Tab | 作用 |
| --- | --- |
| 仪表盘 | 查看当前筛选结果的整体经营总览 |
| 报表 | 查看当前筛选结果按主维度分组后的汇总列表，并继续进入拆解和明细 |

默认进入哪个 Tab 可按产品偏好决定。作为经营看板，“仪表盘”更适合作为默认首屏；作为报表工具，“报表”更适合延续当前用户路径。第一版可以默认展示“仪表盘”，突出新增能力。

## 5. 仪表盘 Tab

仪表盘第一版只展示总览指标卡片，不展示图表。

### 5.1 展示指标

仪表盘展示 10 个指标：

| 指标 | 说明 |
| --- | --- |
| 上报业绩 | 当前筛选结果下的上报业绩总和 |
| 确认业绩 | 当前筛选结果下的确认业绩总和 |
| 成交单量 | 当前筛选结果下的成交记录数 |
| 成交客户数 | 当前筛选结果下的成交客户去重数 |
| 新诊业绩 | 当前筛选结果下新诊成交的上报业绩总和 |
| 新诊单量 | 当前筛选结果下新诊成交记录数 |
| 新诊客户数 | 当前筛选结果下新诊成交客户去重数 |
| 复购业绩 | 当前筛选结果下复购成交的上报业绩总和 |
| 复购单量 | 当前筛选结果下复购成交记录数 |
| 复购客户数 | 当前筛选结果下复购成交客户去重数 |

这些指标都基于当前全局筛选后的数据计算，不受主维度影响。

### 5.2 布局

指标以卡片网格展示。每张卡片包含：

- 指标名称
- 指标数值

金额类指标沿用现有金额格式；数量类指标展示纯数字。

第一版不需要在卡片上增加趋势、同比、环比或说明入口。指标解释仍沿用现有指标 tooltip 能力即可。

### 5.3 后续图表空间

仪表盘可以在指标卡片下方预留后续扩展位置，但第一版不实现具体图表。

后续可迭代方向包括：

- 排名图
- 柱状图
- 趋势图
- 结构占比图

这些方向只作为页面架构预留，不进入本次实现范围。

## 6. 报表 Tab

报表 Tab 承载现有能力：

- 主维度选择
- 主维度汇总列表
- 业绩拆解抽屉
- 业绩明细抽屉

“主维度”放在报表 Tab 内，位置建议在列表上方，作为报表工具栏的一部分。

主维度切换只影响报表列表如何聚合展示，不影响仪表盘的总览指标。

## 7. 数据流

数据流保持简单：

1. 全局筛选区产生筛选条件。
2. 系统基于筛选条件得到同一批 filtered records。
3. 仪表盘基于 filtered records 计算整体总览指标。
4. 报表基于 filtered records 和当前主维度计算主维度汇总行。
5. 抽屉拆解和明细继续基于 filtered records、主维度和当前选中行计算。

核心语义：

- 筛选项决定统计范围。
- 仪表盘展示统计范围的整体总计。
- 主维度决定报表分组。
- 报表行进入拆解和明细。

## 8. 空状态

当筛选后没有数据时：

- 仪表盘指标卡片展示 0 或对应空数值，不隐藏整个仪表盘。
- 报表列表展示现有空表格状态。
- 抽屉不需要额外变化。

## 9. 验收标准

- 页面存在“仪表盘 / 报表”两个 Tab。
- 全局筛选项同时影响仪表盘指标和报表列表。
- 主维度选择只出现在报表 Tab 内。
- 仪表盘展示 10 个总览指标。
- 切换主维度不会改变仪表盘总览指标。
- 报表 Tab 保留现有列表、业绩拆解和业绩明细能力。
- 本次不出现未实现的排名图、柱状图或趋势图。
