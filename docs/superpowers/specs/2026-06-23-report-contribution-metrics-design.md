# 业绩报表：三项贡献指标

## 目标

在业绩报表主列表与业绩拆解列表中，新增用于说明当前行在当前筛选结果中结构占比的三项指标：业绩贡献、成交单量贡献、成交客户贡献。

## 范围

- 包含：业绩报表主列表、业绩拆解列表、对比期展示、指标聚合、字段前缀和自动化测试。
- 不包含：转化漏斗报表、业绩明细抽屉、筛选条件和其他既有指标的排序调整。

## 计算口径

三项指标均基于当前已应用筛选条件的数据集计算。

| 指标 | 分子 | 分母 |
| --- | --- | --- |
| 业绩贡献 | 当前行上报业绩 | 当前筛选条件下的汇总上报业绩 |
| 成交单量贡献 | 当前行成交单量 | 当前筛选条件下的汇总成交单量 |
| 成交客户贡献 | 当前行成交客户数 | 当前筛选条件下的汇总成交客户数 |

- 主列表的分母是当前筛选结果在主维度分组前的汇总值。
- 业绩拆解列表的分母是该拆解表所属主行在拆解维度分组前的汇总值。
- 分母为 0 时，该贡献指标无定义，显示 `—`。

## 展示规则

- 三项指标固定紧跟在“客单价”之后，顺序为：`客单价 → 业绩贡献 → 成交单量贡献 → 成交客户贡献`。
- 主列表与业绩拆解列表复用同一组指标定义、顺序和单元格渲染逻辑。
- 无论客户范围和成交类型是否为“全部”，三项指标始终展示；“全部 + 全部”时列名不带前缀。
- 筛选客户范围或成交类型后，三项指标沿用现有规则添加前缀。例如：`新客新诊业绩贡献`、`新客新诊成交单量贡献`、`新客新诊成交客户贡献`。

## 对比期

选择对比时间时，三项贡献指标与其他业绩指标一致：

1. 首行显示本期贡献率。
2. 次行显示本期贡献率相对对比期贡献率的增减幅：`(本期贡献率 - 对比期贡献率) ÷ 对比期贡献率`。

对比期贡献率无定义、为 0，或缺少可匹配的对比行时，次行显示 `—`。

## 实现边界

在现有业绩报表的贡献值聚合和指标定义中完成最小改动：保留现有三项占比的计算字段，将其改为始终展示的“贡献”文案，并复用现有对比数据计算与双行单元格展示。主列表和业绩拆解不维护独立逻辑。

## 验收与测试

- 主列表与业绩拆解列表均显示三项贡献列，且紧跟客单价。
- 主列表按当前筛选结果汇总为分母；拆解列表按所属主行汇总为分母。
- 三项贡献的分母为 0 时显示 `—`。
- 全部筛选显示无前缀列名；新客/老客和新诊/复购筛选正确添加前缀。
- 选择对比时间时，三项贡献正确显示相对对比期的增减幅；无有效对比值时显示 `—`。
