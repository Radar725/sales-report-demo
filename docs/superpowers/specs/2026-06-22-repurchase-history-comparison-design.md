# 复购历史占比数据对比设计

**日期：** 2026-06-22
**范围：** 仅业绩报表主表与维度拆解表中的复购历史占比字段

## 1. 目标

为下列三个“复购历史占比”增加对比期数据对比，用于判断当期复购对历史沉淀的激活强度是否变强：

1. 复购客户历史占比
2. 复购单量历史占比
3. 复购业绩历史占比

不改变转化漏斗、业绩明细、基础复购指标或既有三项普通占比的口径。

## 2. 统计口径

每个期间均使用“截至该期间期末”的累计复购数据，避免用未来数据作为过去期间的分母。

| 指标 | 本期值 | 对比期值 |
| --- | --- | --- |
| 复购客户历史占比 | 本期复购客户数 ÷ 截至本期期末累计复购客户数 | 对比期复购客户数 ÷ 截至对比期期末累计复购客户数 |
| 复购单量历史占比 | 本期复购单量 ÷ 截至本期期末累计复购单量 | 对比期复购单量 ÷ 截至对比期期末累计复购单量 |
| 复购业绩历史占比 | 本期复购业绩 ÷ 截至本期期末累计复购业绩 | 对比期复购业绩 ÷ 截至对比期期末累计复购业绩 |

对比增减幅沿用业绩报表已有规则：

`(本期历史占比 - 对比期历史占比) ÷ 对比期历史占比`

对比期占比为 0、无对应维度行，或任一分母为 0 时，相应不可计算结果展示 `—`。

## 3. 筛选与聚合边界

每个期间分别派生两份记录集：

1. **期间复购数据**：使用该期间的完整筛选条件，并固定成交类型为“复购”。
2. **截至期末累计复购数据**：保留部门、咨询师、渠道、项目、客户池、城市、机构和客户范围；成交类型固定为“复购”；移除开始日期，只保留“成交日期不晚于该期间结束日”。

本期和对比期使用各自的累计分母，不共用历史基准。

“新客/老客”筛选继续保留在两份记录集中，并按各自期间判断客户创建时间。因此，新客或老客的复购历史占比均只衡量该人群内部的复购动能。

主表按当前主维度计算；维度拆解表先限定主维度行，再按拆解维度计算。累计分母必须与分子的主维度、拆解维度一致。

## 4. 展示与说明

未开启对比时间时，三个字段保持单行本期值展示。

开启对比时间后，指标单元格采用双行布局：

1. 首行显示本期历史占比；
2. 次行显示相对对比期的增减幅。

上涨显示绿色 `↑`，下降显示红色 `↓`，持平显示中性灰，无法计算显示 `—`。

指标说明统一为：**当期复购在截至期末累计复购中的占比，用于衡量复购动能。**

## 5. 验收标准

1. 本期和对比期均以各自期末截断累计复购分母，不使用未来数据。
2. 主表与维度拆解均按相同维度匹配分子和累计分母。
3. 本期历史占比和对比期历史占比均正确展示，增减幅遵循既有相对变化规则。
4. 新客、老客及其他业务筛选在本期和对比期的分子、分母中均保持一致。
5. 分母为 0、对比期占比为 0 或无匹配行时显示 `—`。
6. 未开启对比时间时，三个字段的现有单行展示和其他报表功能不受影响。
