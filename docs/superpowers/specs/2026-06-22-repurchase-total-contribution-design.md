# 复购占总复购比字段设计

**日期：** 2026-06-22  
**范围：** 仅业绩报表主表与维度拆解表

## 1. 目标

当业绩报表的“成交类型”为“复购”时，新增三个用于衡量当期复购对历史复购累计贡献的比例字段：

1. 复购客户占总复购比
2. 复购单量占总复购比
3. 复购业绩占总复购比

字段只影响业绩报表；不改漏斗报表，也不改变现有基础指标与三项既有占比的计算方式。

## 2. 统计口径

### 2.1 分子

分子来自当前统计时间内的复购成交记录，并保留用户当前设置的全部筛选条件：客户统计范围、部门、咨询师、渠道分类、渠道、项目分类、项目、客户池、城市与机构。

### 2.2 分母

分母是同一维度下的历史复购累计数据：

- 忽略“统计时间”，即不受当前 `dateRange` 限制；
- 成交类型固定为“复购”；
- 保留客户统计范围及所有其他业务筛选；
- 主表按当前主维度聚合；维度拆解表先限定当前主维度行，再按拆解维度聚合。

因此：

| 字段 | 计算公式 |
| --- | --- |
| 复购客户占总复购比 | 当前期复购成交客户数 ÷ 该维度历史复购成交客户数 |
| 复购单量占总复购比 | 当前期复购成交单量 ÷ 该维度历史复购成交单量 |
| 复购业绩占总复购比 | 当前期复购上报业绩 ÷ 该维度历史复购上报业绩 |

分母为 0 时，字段值为不可用并展示 `—`。

## 3. 展示规则

- 仅当成交类型为“复购”时展示三列。
- 三列位于现有全部指标列之后、操作列之前；主表与维度拆解表顺序一致。
- 列头只根据客户统计范围添加前缀：
  - 全部：`复购客户占总复购比`、`复购单量占总复购比`、`复购业绩占总复购比`；
  - 新客：分别加“新客”前缀；
  - 老客：分别加“老客”前缀。
- 不再追加成交类型前缀，避免“复购复购客户…”等重复表述。
- 三列可按当前期比例排序。
- 成交类型为“全部”或“新诊”时，三列完全不展示。

## 4. 对比期

三个新字段不参与数据对比：

- 开启对比时间后仍只展示当前期比例；
- 不计算、不存储，也不渲染环比/同比变化值；
- 已有七项业绩报表指标的对比展示保持不变。

## 5. 数据流与实现边界

页面层从当前筛选派生一份历史复购基准筛选：移除统计时间、强制成交类型为复购、保留其余筛选。该基准记录集同时传给主表和维度拆解，领域层再按各自分组范围计算三项比例。

为保持范围最小，改动限于业绩报表的筛选派生、报表行聚合、列定义与相应测试；不改 mock 漏斗数据和既有无关文件。

## 6. 验收与测试

1. 历史基准忽略统计时间，保留客户统计范围及其余筛选，并固定复购成交类型。
2. 主表和维度拆解表均按各自维度正确计算三项比例。
3. 历史分母为 0 时显示 `—`。
4. 复购筛选显示三列及正确的新客/老客前缀；全部、新诊筛选不显示。
5. 对比时间开启时，新字段无变化值；现有字段仍展示变化值。
