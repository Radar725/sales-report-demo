# 转化漏斗报表：有效客户统计口径设计

日期：2026-06-22

## 1. 目标与范围

转化漏斗报表不再提供“客户类型”筛选。报表保留全部录单客户作为录单质量基数，并新增有效客户相关指标；业务转化阶段仅统计有效客户。

本次仅调整转化漏斗报表的筛选项、指标计算、列表字段、演示数据和测试。不改业绩报表、主维度、维度拆解规则、客户当前状态口径、真实接口或导出能力。

## 2. 筛选与统计范围

- 移除筛选栏中的“客户类型”控件及其对应筛选状态。
- 保留录单时间、对比时间、咨询师和渠道筛选。
- 当前期、对比期均先按录单时间、咨询师和渠道筛选出全部录单客户。
- 维度拆解沿用主表已锁定的主维度值，并使用相同的统计范围和指标口径。

## 3. 指标口径

### 3.1 客户基数

| 字段 | 计算 |
| --- | --- |
| 录单客户数 | 符合筛选条件的全部去重录单客户数，包含有效与无效客户 |
| 有效客户数 | 上述录单客户中标记为有效的去重客户数 |
| 有效客户率 | 有效客户数 ÷ 录单客户数 |

### 3.2 阶段指标

已加微、已派单、已邀约、已到院、已成交、已复购客户数，均只在有效客户集合中按现有“当前状态累计到达阶段”的规则计算。无效客户不计入任何阶段客户数。

各阶段率均以有效客户数为分母：

- 加微率 = 已加微有效客户数 ÷ 有效客户数；
- 派单率 = 已派单有效客户数 ÷ 有效客户数；
- 邀约率 = 已邀约有效客户数 ÷ 有效客户数；
- 到院率 = 已到院有效客户数 ÷ 有效客户数；
- 成交率 = 已成交有效客户数 ÷ 有效客户数；
- 复购率 = 已复购有效客户数 ÷ 有效客户数。

任一比例的分母为 0 时展示 `—`。阶段累计关系仍保证阶段客户数及阶段率从前向后不增加。

## 4. 列表展示

主表和维度拆解抽屉使用相同字段与顺序：

1. 录单客户数
2. 有效客户数
3. 已加微客户数
4. 已派单客户数
5. 已邀约客户数
6. 已到院客户数
7. 已成交客户数
8. 已复购客户数
9. 有效客户率
10. 加微率
11. 派单率
12. 邀约率
13. 到院率
14. 成交率
15. 复购率

对比时间开启时，每一列继续显示当前值及相对于对比期的变化；当前期和对比期的指标计算完全一致。

## 5. 实现边界

- 移除漏斗筛选状态中的客户类型字段、默认值及过滤逻辑。
- 计算层先得到全部录单客户，再从中派生有效客户集合：录单客户数基于前者，其余新增及阶段指标按本设计分别取数。
- 保留无效客户演示数据，确保有效客户率可反映录单质量；补充至少一组无有效客户的样本以验证零分母展示。
- 不修改客户当前状态到漏斗阶段的映射，也不引入历史状态快照或阶段动作时间。

## 6. 验证范围

测试至少覆盖：

1. 录单客户数包含无效客户，有效客户数与有效客户率计算正确；
2. 无效客户不计入任一阶段客户数及阶段率；
3. 阶段率均以有效客户数为分母，且有效客户数为 0 时为 `null` 并展示 `—`；
4. 当前期、对比期、主表与维度拆解使用同一口径；
5. 筛选栏不再出现客户类型，重置操作不再依赖该字段；
6. 主表与维度拆解的字段顺序一致，并符合本设计的 B 布局。
